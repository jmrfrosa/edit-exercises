Partindo do nosso exercício 1.1 (lista de utilizadores), vamos adicionar várias funcionalidades à aplicação:

1. Vamos adicionar um formulário que permite criar novos utilizadores.
2. Queremos, também, um botão "Apagar" em cada utilizador, que apaga o utilizador da lista.
3. Por último, cada um dos nossos utilizadores tem agora uma nova propriedade "isAdmin". Cria dois novos botões que servem para atribuir e retirar capacidades de administrador a cada utilizador.
